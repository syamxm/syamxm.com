// about

Built by hand, on purpose.

I run a Debian homeserver behind Tailscale and a Cloudflare Tunnel — zero open inbound ports — and everything I build deploys to it through a fail-closed pipeline: php-l/semgrep/gitleaks before build, Trivy on the filesystem and image after, deploy only when main is green.

This site is part of that: hand-written HTML and CSS, self-hosted fonts, default-src 'self' CSP, no third-party origins.
